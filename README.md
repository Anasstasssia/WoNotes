# WoNotes

Приложение, где заметки живут рядом с календарём цикла — потому что обе вещи нужны каждый день, и обе вечно теряются по разным углам телефона.

Идея простая: не нужно переключаться между приложением для заметок и трекером цикла. WoNotes объединяет их в одном месте. Пишешь мысли, списки, планы — и тут же видишь, какой сейчас день цикла, когда ждать следующих, и где примерно фертильное окно. Всё хранится локально на устройстве, зашифровано, без регистрации и облаков.

---

## Функционал

**Заметки**
- Создание, редактирование и удаление заметок
- Заметки сгруппированы по датам — листается как дневник
- Поиск по заголовку и тексту
- Поделиться заметкой через любое приложение

**Трекер цикла**
- Отметка дней менструации прямо на календаре
- Расчёт средней длины цикла и длительности менструации
- Прогноз фертильного окна и предполагаемого дня овуляции
- Статистика: текущий день цикла, регулярность, история предыдущих циклов
- Данные хранятся локально и зашифрованы

## Установка на Android

Приложение поддерживает **Android 8.0 и выше** (API 26+).

### Способ 1 — через APK-файл (прямо на телефоне)

1. Скачай APK-файл на телефон
2. Открой файл через файловый менеджер — система попросит разрешение на установку из неизвестных источников
3. Зайди в **Настройки → Безопасность → Установка неизвестных приложений** и разреши установку для своего файлового менеджера или браузера, через который скачивала
4. Вернись к APK-файлу и нажми **Установить**

> На разных прошивках путь к настройке может немного отличаться, но называется примерно одинаково — «Неизвестные источники» или «Установка приложений».

### Способ 2 — сборка из исходников

Нужен [Flutter](https://flutter.dev/docs/get-started/install) версии 3.10+.

```bash
git clone https://github.com/your-username/wonotes.git
cd wonotes
flutter pub get
flutter run
```

Собрать APK:

```bash
flutter build apk --release
```

Готовый файл появится по пути `build/app/outputs/flutter-apk/app-release.apk`.
